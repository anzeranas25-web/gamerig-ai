# GameRig AI Netlify Drop Website

This is a static prototype website for **GameRig AI**.

## Files

- `index.html` - full landing page and product pitch
- `style.css` - responsive dark gamer UI
- `script.js` - interactive demo logic for Builder, PS5 Destroyer, and Build Battles

## How to deploy on Netlify Drop

1. Unzip the folder.
2. Go to Netlify Drop.
3. Drag the whole `gamerig-ai-netlify` folder into Netlify.
4. Netlify will publish it as a static site.
